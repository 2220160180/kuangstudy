# Ajax

## 1、什么是Ajax

AJAX= Asynchronous  JavaScript And XML(异步的JavaScript和xml)

Ajax是一种无须重新加载整个网页情况下，能够更新部分网页的技术.

Ajax不是一种新的编程语言，而是一种用于创建更好更快以及交互性更强的Web应用程序的技术。

增强B/S的体验性，未来的主流，并且会爆发式的持续增长! 

产品链： H5+网页+客户端+手机端(Android、ios)、+小程序
